Fanmade website to track progression on tauri.

https://tauriprogress.github.io/

To run locally:

1. install [node](https://nodejs.org/)
2. install [git](https://git-scm.com/downloads)
3. run: git clone https://github.com/tauriprogress/tauriprogress.github.io.git tauriprogress
4. run: cd tauriprogress
5. run: npm install
6. run: npm run dev
